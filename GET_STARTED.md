## 🚀 How to Use

### Option 1: CLI Tool (Command Line)

```bash
python problem_reporter.py
```

- Interactive 7-step form
- Generate professional reports
- Export to text/JSON
- No server required
- Perfect for individuals

### Option 2: Website Application

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run server
python server.py

# 3. Visit: http://localhost:5000
```

- Beautiful responsive interface
- Multi-user support
- Real-time dashboard
- Analytics & statistics
- Report tracking
- Comments & updates
